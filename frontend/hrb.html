Le prélèvement sur compte bancaire SEPA

  • Le paiement par prélèvement SEPA est la nouvelle norme de paiement sécurisé sur Internet

    PureSanté Éditions applique la nouvelle norme européenne SEPA (Single Euro Payment Area) pour les prélèvements sur compte bancaire.

    PureSanté Éditions vous garantit le remboursement immédiat et sans frais sur simple demande.

    Avec le prélèvement SEPA, votre banque vous rembourse elle-aussi sans condition jusqu'à 8 semaines après votre paiement (les banques n'offrent pas cette garantie pour les paiements par chèque ou par carte bancaire).

  • Votre commande est facilitée

    Pour effectuer le prélèvement SEPA, vous n'avez pas besoin de divulguer votre numéro de carte de crédit. Il vous suffit d'indiquer dans le formulaire ci-dessous votre numéro IBAN ainsi que votre code SWIFT/BIC. Ces informations figurent sur votre Relevé d'Identité Bancaire (RIB) que vous trouverez sur votre relevé de compte, à la fin de votre chéquier, ou sur votre interface bancaire sur Internet.

     

  • La nouvelle norme de paiement sur Internet

    Aussi le paiement par prélèvement sur compte bancaire SEPA remplace dorénavant le paiement par carte bancaire.

    Si vous avez déjà souscrit un abonnement auprès de PureSanté Éditions par carte bancaire, celui-ci reste bien évidemment valable.
